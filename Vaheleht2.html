Pythoni andmetüübid

Sissejuhatus

Pythoni üheks tähtsamaks osaks on muutujad ning andmed, mida muutujad endas talletavad. Andmete liiki nimetatakse programmeerimisel andmetüübiks. Järgnevalt tutvume põhiliste andmetüüpidega ning ka mõne keerukamaga.

Näiteks: rida "x = 4" omistab muutujale x väärtuse täisarvu 4.

Põhilised andmetüübid

Pythonis on kõige aluseks 4 põhilist andmetüüpi. Andmetüüpe on küll rohkem, aga lõppude lõpuks tuginetakse alati neile neljale.
Hea teada: Pythoni käsklus type tagastab antud väärtuse tüübi.

str - string ehk sõne
Kõik mis jääb jutumärkide (" ") või ülakomade (' ') vahele.
NB! Ka ujukoma- ja täisarvud loevad sõnena, kui need jätta jutumärkide vahele.

Antud kontekstis on "9" hoopis sõne.

int - integer ehk täisarv
Lihtsalt täisarv (nt 9, 58, 2503)

float - floating point number ehk ujukomaarv
Arv, milles esineb komakoht (nt 4.45, aga ka 6.0)

bool - boolean ehk tõeväärtus
Antud andmetüüp on omamoodi, kuna sellel saab olla täpselt 2 väärtust: True või False.
Ühe elemendi tõeväärtust saab kontrollida käsu bool() abil. Kui element on mingi kindel väärtus, millega saab midagi teha, on tema tõeväärtus True.
Kui element on nt 0, tühi sõne (ehk "") või None, siis on tema tõeväärtus False.

Kuigi tõeväärtust saab kasutada teiste elementide uurimiseks, saab tõeväärtus ise olla ka element.

Tasub ka mainida, et teatud operatsioonid töötavad ainult teatud andmetüüpidega. Rida
x = "Tere!" / "Maailm!"
tagastab errori, kuna kahte sõnet ei saa omavahel jagada.

Küll aga töötab rida
x = "Tere!" + " " + "Maailm!"
sest sõnesid saab omavahel liita.

Keerukamad andmetüübid

Nagu eelnevalt juba mainitud, siis on Pythonis tegelikult rohkem andmetüüpe. Andmetüüpe, mille väärtused talletavad endas veel teisi väärtuseid nimetatakse andmestruktuurideks. Uurime mõnda neist lähemalt:

list - list ehk järjend
Võib sisaldada kuitahes palju elemente, kusjuures elemendid võivad korduda. Järjendi tunneb ära nurksulgude [ ] järgi.
Tühja järjendit saab koostada käsuga järjend = [] ning sellesse saab elemente lisada käsuga järjend.append(element).

Siinkohal tasub mainida, et kui soovite saada järjendist kätte ühte kindlat elementi, siis peab selle esile kutsuma indeksi kaudu.
Python alustab indeksite lugemist nullist.
Nt järjendi järjend = ["talv", "kevad", "suvi", "sügis"] elemendile "talv" vastab indeks 0, elemendile "kevad" indeks 1 jne.

Järjendi elemendi indeksit saab uurida järjend.index(element) käsuga.

set - set ehk hulk
Sarnaneb järjendiga, erandiks see, et hulgas ei saa elemendid korduda ning elemendid ei oma indekseid. Hulga tunneb ära looksulgude järgi { }.
NB! Ka sõnastik kasutab looksulge!
Tühja hulka saab koostada käsuga hulk = set() ning sellesse saab elemente lisada käsuga hulganimi.add(element)

Kuna hulgas indeksid puuduvad, ei ole elementide järjekord oluline

dict - dictionary ehk sõnastik
Koosneb võtmetest ja väärtustest. Tühja sõnastiku saab koostada käsuga sõnastik = {}.
Elementide lisamiseks sõnastikku on vaja defineerida nii võti kui ka väärtus. Nt. Käsklus sõnastik[x] = y määrab võtmele x väärtuse y.

Indeksid sõnastike puhul ei kehti, kõik väärtused saab kätte neile vastavate võtmete kaudu.
Sõnastiku sõnastik = {"võti": "väärtus"} võtme "võti" väärtuse tagastamiseks peame kirjutama print(sõnastik["võti"])

Nagu näha, kõik need andmestruktuurid omistavad ühele muutujale väärtuse, mis omakorda koosneb veel mitmest väärtusest ja need väärtused on veel omakorda sõned, täisarvud või ujukomaarvud.
