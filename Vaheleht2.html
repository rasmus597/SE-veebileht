Pythoni andmetüübid

Pythoni üheks tähtsamaks osaks on muutujad ning andmed, mida muutujad endas talletavad.

Näiteks: rida "x = 4" omistab muutujale x väärtuse täisarvu 4.

Põhilised andmetüübid

Pythonis on kõige aluseks 3 põhilist andmetüüpi. Andmetüüpe on küll rohkem, aga lõppude lõpuks tuginetakse alati neile kolmele. Veel üheks põhiliseks andmetüübiks on kompleksarvud, mida me siin lehel ei käsitle.
Hea teada: Pythoni käsklus type tagastab antud väärtuse tüübi.

str - string ehk sõne, kõik mis jääb jutumärkide ("") või ülakomade ('') vahele.
NB! Ka ujukoma- ja täisarvud loevad sõnena, kui need jätta jutumärkide vahele.

int - integer ehk täisarv (nt 3, 19, 206)

float - floating point number ehk ujukomaarv, st arv, milles esineb komakoht (nt 4.45, aga ka 6.0)

Tasub ka mainida, et teatud operatsioonid töötavad ainult teatud andmetüüpidega. Rida x = "Tere!" / "Maailm!" tagastab errori, kuna kahte sõnet ei saa omavahel jagada.

Keerukamad andmetüübid

Nagu eelnevalt juba mainitud, siis on Pythonis tegelikult rohkem andmetüüpe. Uurime mõnda neist lähemalt:

str - string ehk sõne, kõik mis jääb jutumärkide ("") või ülakomade ('') vahele.
NB! Ka ujukoma- ja täisarvud loevad sõnena, kui need jätta jutumärkide vahele.

int - integer ehk täisarv (nt 3, 19, 206)

dict - dictionary ehk sõnastik. Koosneb võtmetest ja väärtustest. Nt. Käsklus sõnastik[x] = y määrab võtmele x väärtuse y.

Kõik need andmetüübid omistavad ühele muutujale väärtuse, mis omakorda koosneb veel mitmest väärtusest.
