Pythoni andmetüübid

Pythoni üheks tähtsamaks osaks on muutujad ning andmed, mida muutujad endas talletavad.

Näiteks: rida "x = 4" omastab muutujale x väärtuse täisarvu 4.
Põhilised andmetüübid

String

str - string - sõne, kõik mis jääb jutumärkide ("") või ülakomade ('') vahele.
NB! Ka ujukoma- ja täisarvud loevad sõnena, kui need jätta jutumärkide vahele.

Integer

int - integer - täisarv (nt 3, 19, 206)

Floating Point Number

float - floating point number - ujukomaarv, st arvus esineb komakoht (nt 4.45, aga ka 6.0)
