Funktsioonid ja tsüklid

1. Funktsioonid

Funktsioon on korduvkasutatav koodiplokk, mis täidab konkreetset ülesannet. See aitab muuta koodi loetavamaks, organiseeritumaks ja vähendada koodi kordamist.

Funktsiooni defineerimine

Funktsioon defineeritakse märksõnaga def, millele järgneb funktsiooni nimi ja sulgudes parameetrid. 

# Funktsiooni defineerimine ilma parameetriteta
def tervita():
    print("Tere tulemast Pythoni maailma!") # Väljastab sõne

# Funktsiooni defineerimine parameetritega
def liida_arvud(a, b):
    tulemus = a + b
    return tulemus # Tagastab väärtuse

Funktsiooni väljakutsumine

Funktsiooni käivitamiseks tuleb see välja kutsuda selle nime ja vastavate argumentidega. 

tervita() 

summa = liida_arvud(15, 7)
print(f"Summa on: {summa}")

2. Tsüklid

Tsüklid võimaldavad korduvate ülesannete automatiseerimist, käivitades koodiploki korduvalt, kuni teatud tingimus on täidetud.

For-tsükkel

for-tsüklit kasutatakse tavaliselt järjendite, sõnede, hulkadega seotud toimingute jaoks või koodiploki kordamiseks määratud arv kordi (range() funktsiooniga). 

# For-tsükkel järjendi jaoks
puuviljad = ["õun", "banaan", "kirss"]
for puuvili in puuviljad:
    print(f"Mulle meeldib {puuvili}.")

# For-tsükkel numbrilises vahemikus kordamiseks
for i in range(3): # kordab 3 korda (0, 1, 2)
    print(f"Kordus number {i + 1}")

While-tsükkel

while-tsükkel kordab koodiplokki seni, kuni antud tingimus on tõene (True). 

loendur = 0
while loendur < 5: 
    print(f"Praegune loendur: {loendur}")
    loendur += 1

Tsükli kontrollvood

  • break: Katkestab tsükli koheselt.
  • continue: Jätab vahele tsükli käesoleva osa ja jätkab järgmise osaga.
